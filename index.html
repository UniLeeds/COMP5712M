    Programming for Data Science¶

    This learning resource contains all the lessons, activities and exercises you need to complete this module. You can navigate between the units as you progress through the module, allowing you to easily refer back to previous lessons if you need.

    You can navigate this resource in three ways:

    1. The navigation bar at the top of this document allows you to move between units.

    2. The left-hand navigation bar allows you to move between lessons and activities.

    3. The right-hand navigation bar allows you to move around within a lesson or activity.

    You may also find it useful to use the search function within this resource to find what you’re looking for. This will allow you to search all the materials across the whole module.

    Within the lessons you will also find links to the Jupyter notebook files. These contain the exercises that give you an opportunity to practise and test what you’ve been shown in the lessons.

    At various points throughout the lessons, you will be directed back to Minerva to participate in discussions. Please take the opportunity to share the interesting and challenging aspects of this module in the discussions with your peers.

    Don’t forget to regularly return to the Minerva portal to view module announcements, changes to webinar times and to follow the assessment schedule.

    To start, select a unit from the top navigation.
